# 幹話產生器

只要選擇對象，就可以產生適當的幹話


## 現有功能
- 選擇對象，就可以產生相對應的幹話


## 使用工具
- Node.js: 13.0.1
- Express: 4.17.1
- Express-Handlebars: 5.1.0
- nodemon: 2.0.4

## 安裝
1.開啟終端機(Terminal) cd 到存放專案本機位置並執行:

```
git clone https://github.com/windsailg/ac_restaurant_list.git
```


2.初始化安裝套件

```
npm install  //安裝套件
```

```
npm run dev  //啟動程式
```


3.Terminal 顯示以下資訊


```
Express is listening on localhost:3000 
//伺服器已經成功運作於 http://localhost:3000
```
